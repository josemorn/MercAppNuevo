# MercApp - Frontend

Aplicación SPA con Vue 3, Vite y Router.

## Instalación
```bash
npm install
```

## Desarrollo
```bash
npm run dev
```

Accede a http://localhost:5173

## Build
```bash
npm run build
```

## Variables de Entorno

- `VITE_API_URL`: URL del backend API

Copia `.env.example` a `.env.local` y personaliza según necesites.
