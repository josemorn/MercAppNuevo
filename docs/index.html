MercApp - Sistema de Ecommerce

Aplicación SPA con Vue 3, API propia y base de datos MongoDB

Arquitectura de Despliegue

  • Frontend: Vue 3 + Vite en Netlify
  • Backend: Express + Node.js en Railway
  • Database: MongoDB Atlas (M0 Free)
  • HTTPS: Automático en ambas plataformas

Enlaces Importantes

  • Frontend (SPA): https://mercapp.netlify.app
  • Backend API: https://mercapp-api.up.railway.app
  • Health Check: /health
  • Repositorio GitHub: GitHub

Documentación de Endpoints

Método Ruta Descripción
GET /api/products Obtiene lista de productos
GET /api/products/:id Obtiene un producto por ID
POST /api/products Crea un nuevo producto
PUT /api/products/:id Actualiza un producto
DELETE /api/products/:id Elimina un producto
GET /api/categories Obtiene todas las categorías

Stack Técnico

  • Frontend: Vue 3, Vite, Vue Router
  • Backend: Node.js, Express, MongoDB Driver
  • Database: MongoDB Atlas (M0)
  • Hosting: Railway (API), Netlify (Frontend)
  • HTTPS: Automático
  • CORS: Configurado

Ejecución Local

Backend

cd backend
npm install
npm run dev
# Servidor en http://localhost:5000

Frontend

cd frontend
npm install
npm run dev
# Aplicación en http://localhost:5173
